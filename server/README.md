# 后端开发依赖组件说明

```bash
# 命令行参数库，文档：https://cobra.golang.ac.cn/
go get -u github.com/spf13/cobra
# 时间日期库，文档：https://carbon.go-pkg.com/zh/usage/default.html
go get -u github.com/dromara/carbon/v2
# Web框架，文档：https://gin-gonic.com/zh-cn/docs/introduction/
go get -u github.com/gin-gonic/gin
# 配置文件读取库
go get -u github.com/spf13/viper
# 日志库
go get -u go.uber.org/zap
# 日志切割
go get -u gopkg.in/natefinch/lumberjack.v2
# 数据库驱动
go get -u gorm.io/driver/mysql
# Redis 数据库驱动
go get -u github.com/redis/go-redis/v9
# JWT 库
go get -u github.com/appleboy/gin-jwt/v3
# 结构体赋值转换
go get -u github.com/jinzhu/copier
# JSON 库
go get -u github.com/bytedance/sonic
```
